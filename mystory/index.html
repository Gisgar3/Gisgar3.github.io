My Story

The first time I was really interested in learning how to program was when my school starting advertising a coding site; which mainly specialized
in JavaScript. I thought the website was pretty decent; had input and output boxes that actually showed the running code too. That was
when I knew I wanted to learn more about programming. I went home after school for days and worked on learning JavaScript. I
still have notebooks filled with my old learning code. After finishing my full tutorial, I started my first project! Minecraft was starting
to become more popular at the time, so modding the game seemed like a good idea to get my thoughts out there. My first(and only) mod on
Minecraft was a chocolate mod. It was very basic, it contained only a few food items and blocks, obviously made out of chocolate! I was very
proud of myself, it was my first coding project that worked out pretty well-atleast for awhile. I lost most of my code from a data corruption.
Some code still exists, but it would be useless these days, the code is very outdated. Now, to the present!

Now in 2017, I specialize in C# software development, and also manage a small organization called Stable Programming. If you want to
